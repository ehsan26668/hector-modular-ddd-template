### 🚀 v1.0.0 – Initial Stable Release

This is the first stable release of the **Modular DDD .NET Template**. This project provides a production-grade modular monolith architecture built with .NET.

#### Highlights

- **Modular Architecture:** Strict boundaries with decoupled feature modules.
- **Internal Mediator:** CQRS implementation with pipeline behaviors.
- **Transactional Outbox:** Guaranteed event delivery for domain events.
- **Inbox Pattern:** Idempotent event processing for robust integration.
- **Strongly Typed IDs:** Automatic EF Core mapping to prevent primitive obsession.
- **Correlation Propagation:** End-to-end tracing across the system.
- **Architecture Guard Tests:** Enforced project structure using ArchUnitNET.

This template is designed to serve as a reference architecture for scalable, maintainable enterprise systems.
