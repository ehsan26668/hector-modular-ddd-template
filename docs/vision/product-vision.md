# Product Vision: Hector Modular Monolith Template

## Introduction

این پروژه یک Starter Kit مدرن برای توسعه سیستم‌های سازمانی با استفاده از .NET 10 است. هدف اصلی، ارائه زیرساختی است که سرعت توسعه را بالا برده و در عین حال نظم معماری (Architectural Integrity) را حفظ کند.

## Core Values

1. **Maintainability**: کدها باید به گونه‌ای باشند که ۶ ماه بعد به راحتی قابل فهم و تغییر باشند.
2. **Module Autonomy**: هر ماژول باید تا حد امکان مستقل باشد تا در آینده پتانسیل تبدیل به Microservice را داشته باشد.
3. **Developer Experience (DX)**: استفاده از ابزارهای مدرن (مثل slnx) و ساختار شفاف برای راحتی توسعه‌دهنده.
4. **Test-Driven Excellence**: هیچ کدی بدون تست وارد Core فریمورک نمی‌شود.

## Target Architecture

- Modular Monolith
- Domain-Driven Design (DDD)
- Clean Architecture
- Outbox Pattern for Reliable Messaging
